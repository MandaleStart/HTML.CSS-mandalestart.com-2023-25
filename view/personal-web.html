Proyecto 2: Sitio Web Personal

Un proyecto de desarrollo web para una página personal implica crear un sitio que sirva como plataforma para compartir información sobre uno mismo, intereses y actividades.

  1. Requerimientos: Objetivos, contenido y funcionalidades del sitio.
  2. Diseño UI/UX: Interfaz atractiva, colores, tipografía e identidad personal.
  3. Desarrollo: HTML, CSS, JS, Bootstrap y CMS si aplica.
  4. Pruebas: Validación de navegación y funcionamiento responsive.
  5. Lanzamiento: Deploy y seguimiento de mejoras según comentarios.
  6. Promoción: SEO, redes sociales y difusión del contenido.
