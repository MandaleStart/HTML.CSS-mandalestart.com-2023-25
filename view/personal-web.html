Proyecto 2: Sitio Web Personal

Este proyecto consiste en un sitio web personal diseñado para mostrar información, proyectos, redes sociales y contenido relevante sobre el usuario.

  1. Requerimientos: Objetivos, contenido y funcionalidades deseadas.
  2. Diseño UI/UX: Colores, identidad personal, accesibilidad y estructura.
  3. Desarrollo: HTML, CSS, JS, Bootstrap y librerías asociadas.
  4. Pruebas: Compatibilidad móvil y experiencia de usuario.
  5. Lanzamiento: Deploy, SEO y análisis de visitas.
  6. Promoción: Redes sociales, refuerzo visual y branding.
