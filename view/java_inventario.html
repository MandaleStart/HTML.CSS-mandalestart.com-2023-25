Sistema de Gestión de Libros y Equipos

Este software fue desarrollado en Java con base de datos PostgreSQL para la gestión completa de libros, equipos informáticos y préstamos internos.

Características principales

  • ✔ Registro y gestión de libros con ISBN, autores y categorías.
  • ✔ Inventario de equipos informáticos (PC, notebooks, routers, etc.).
  • ✔ Gestión de préstamos a usuarios internos.
  • ✔ Control de stock y alertas por equipo en reparación.
  • ✔ Reportes automáticos exportables a PDF.
  • ✔ Integración con PostgreSQL mediante JDBC.

Tecnologías utilizadas

Java, JDBC, PostgreSQL, Swing/JavaFX, MVC, PostgreSQL + pgAdmin.
