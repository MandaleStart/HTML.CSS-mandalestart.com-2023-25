Sistema de Gestión de Libros y Equipos

Este software fue desarrollado en Java con base de datos PostgreSQL, permitiendo gestionar libros, equipos informáticos y préstamos internos.

Características principales

  • ✔ Registro y gestión de libros con ISBN y autores.
  • ✔ Inventario de equipos informáticos.
  • ✔ Gestión de préstamos internos.
  • ✔ Control de stock y alertas de reparaciones.
  • ✔ Reportes automáticos en PDF.
  • ✔ Conexión JDBC con PostgreSQL.
